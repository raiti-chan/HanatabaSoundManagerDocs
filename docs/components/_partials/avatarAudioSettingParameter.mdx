#### ゲイン
アバター内に組み込まれているオーディオソースの音量を制限できます。
単位はゲインです。

- デフォルト : 10
- 範囲 : 0 ~ 10

#### 減衰が始まる距離
アバターの音声の減衰が始まる距離をメートルで指定できます。
なお、これは音声ソースの `maxDistance` と比較され、より小さい方の値が使用されます。

- デフォルト : 0
- 範囲 : 0 ~ 無制限

#### 聞こえる最大範囲
アバターの音声が届く最大距離をメートルで設定します。
この値を下げると、他のプレイヤーのアバターの音が届く距離を短くでき、0に設定するとそのプレイヤーを事実上「ミュート」できます。
この値はオーディオソースの `maxDistance` と比較され、より小さい方が採用されます。

- デフォルト : 40
- 範囲 : 0 ~ 無制限

#### 音源の大きさ
アバターの音源は通常、点音源としてシミュレートされますが、この値を変更することで、音源がより広い範囲から聞こえるようにできます。
:::warning[警告]
この設定は慎重に使用すべきであり、主に、通り過ぎる際に「大きく」聞こえる必要がある遠方の音源向けに設計されています。

何が起こるか理解していない限り、この値はゼロのままにしておいてください。

`Volumetric Radius`の値は、常に`Avatar AudioFarRadius` よりも小さくする必要があります。デフォルトは0メートルです。
:::

- デフォルト : 0
- 範囲 : 0 ~ 無制限

#### 空間音声として強制する
この設定がオンの場合、オーディオソースに対して空間化が有効になり、 `spatialBlend` が 1 に設定されます。

- デフォルト : オフ

#### カスタム曲線を使用する
オーディオソースで事前に設定されたカスタムカーブを使用するかどうかを設定します。

- デフォルト : オフ
