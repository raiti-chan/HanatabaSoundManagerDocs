:::warning
このコンポーネントは全員のクライアント内で存在しアクティブである必要があります。
クライアント間でオブジェクトの状態に矛盾がある場合、防音自体は動作しますが、音声設定が反映されない可能性があります。
:::
---
## パラメータ説明
#### エリアID
エリアIDです。

:::danger[警告]
同じIDが複数エリアに定義されると、音声設定が正しく反映されません。
必ずエリア毎に一意のIDを指定してください。
:::

:::warning
変更を行うと後述するパススルーエリア設定とズレる可能性があります。
基本途中での変更はおすすめしません。
:::

#### 優先度
エリアの優先度を指定できます。
値が大きいエリアが優先されます。
同値の場合、後に侵入したエリアが優先されます。

- デフォルト : 0

#### リモートに対してのみエリアを適用する
リモートに対してのみ、エリアを適用できます。
音声の送信者としてのみエリアを変更したい場合に使用できます。
エリア内の人の聞こえるエリアは変更されません。

> 例 : 聞こえる範囲は変えずに、他エリアへ貫通するマイク等

- デフォルト : オフ

### エリア内の音声設定 - 声設定
エリア内での声の音声設定を指定できます。

<VoiceSettingParameter/>

### エリア内の音声設定 - アバター音声設定
エリア内でのアバター内に組み込まれているオーディオソースの音声設定を指定できます

<AvatarAudioSettingParameter/>

### パススルー設定
ここに指定したエリア内に、このエリアの音声が聞こえるようになります。

#### ブラックリストを使用する
有効化するとパススルー設定がブラックリストになります。

すべてのエリアに対しこのエリアの音声が聞こえるようになり、指定したエリア内のみ聞こえなくなります。

- デフォルト : オフ

#### パススルーエリア設定
このエリア内の音声が聞こえるエリアを指定できます。
