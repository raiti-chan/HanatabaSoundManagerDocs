# PlayerSoundManager

![コンポーネント画像](./PlayerSoundManager.png)

場所 : `Hanataba/SoundManager/[HSM] PlayerSoundManager`

:::danger[注意]
ワールド内に複数配置すると正常に動作しません。
必ず必要1つ配置してください。
:::

:::warning[注意]
動作させるためには専用の子オブジェクトが必要です。

プレハブを使用するか、自動生成ボタンを押し、必要なオブジェクトを生成してください。

プレハブは `Assets/Hanataba/Udon/HanatabaSoundManager/Prefab/PlayerSoundManager.prefab` にあります。
:::
---
## パラメータ説明
#### ワールドに入った初期のミュート時間(秒)
ワールドにJoinした際に初期設定されるミュート時間です。

1 ~ 3 程度で設定することをおすすめします。

:::warning[注意]
0秒に設定すると、同期の関係で、入った瞬間に防音室の音声が聞こえる可能性があります。
:::
### デフォルトエリアの音声設定 - 声設定
エリア内に入ってない通常エリア内の声の音声設定を指定できます。

#### ゲイン

声の音量をデシベルで指定できます。

- デフォルト : 15
- 範囲 : 0 ~ 24

:::warning[注意]
こちらで音声を大きくする場合、音割れを起こす可能性があります。
:::

#### 減衰が始まる距離

声の減衰が始まる距離をメートルで指定できます。

- デフォルト : 0
- 範囲 : 0 ~ 1,000,000

#### 聞こえる最大範囲

声の聞こえる最大範囲をメートルで指定できます。
この範囲以上離れると声が一切聞こえなくなります。

- デフォルト : 25
- 範囲 : 0 ~ 1,000,000

#### 音源の大きさ
通常、プレイヤーの声は点音源としてシミュレートされますが、この値を変更することで、音源がより広い範囲から聞こえているように調整できます。
:::warning[注意]
この設定は慎重に使用すべきであり、主に、通り過ぎる際に「大きく」聞こえる必要がある遠方の音源向けに設計されています。

何が起きるか理解していない限り、この値は0に設定しておいてください。

`音源の大きさ` の値は、常に `聞こえる最大範囲` よりも小さくする必要があります。
:::

- デフォルト : 0
- 範囲 : 0 ~ 1,000

#### ローパスフィルタ
声が少し離れた場所から聞こえる場合、騒がしい環境でも聞き取りやすくするために、その音声はローパスフィルターを通されます。
このフィルターを適用したくない場合は、無効にできます。

例えば、プレイヤーがボイスチャンネルを使って高品質なDJミックスを再生する場合などは、このフィルターをオフにすることをお勧めします。

- デフォルト : 有効

### デフォルトエリアの音声設定 - アバター音声設定
アバター内に組み込まれているオーディオソースの音量を制限できます。

#### ゲイン
アバター内に組み込まれているオーディオソースの音量を制限できます。
単位はゲインです。

- デフォルト : 10
- 範囲 : 0 ~ 10

#### 減衰が始まる距離
アバターの音声の減衰が始まる距離をメートルで指定できます。
なお、これは音声ソースの `maxDistance` と比較され、より小さい方の値が使用されます。

- デフォルト : 0
- 範囲 : 0 ~ 無制限

#### 聞こえる最大範囲
アバターの音声が届く最大距離をメートルで設定します。
この値を下げると、他のプレイヤーのアバターの音が届く距離を短くでき、0に設定するとそのプレイヤーを事実上「ミュート」できます。
この値はオーディオソースの `maxDistance` と比較され、より小さい方が採用されます。

- デフォルト : 40
- 範囲 : 0 ~ 無制限

#### 音源の大きさ
アバターの音源は通常、点音源としてシミュレートされますが、この値を変更することで、音源がより広い範囲から聞こえるようにできます。
:::warning[警告]
この設定は慎重に使用すべきであり、主に、通り過ぎる際に「大きく」聞こえる必要がある遠方の音源向けに設計されています。

何が起こるか理解していない限り、この値はゼロのままにしておいてください。

`Volumetric Radius`の値は、常に`Avatar AudioFarRadius` よりも小さくする必要があります。デフォルトは0メートルです。
:::

- デフォルト : 0
- 範囲 : 0 ~ 無制限

#### 空間音声として強制する
この設定がオンの場合、オーディオソースに対して空間化が有効になり、 `spatialBlend` が 1 に設定されます。

- デフォルト : オフ

#### カスタム曲線を使用する
オーディオソースで事前に設定されたカスタムカーブを使用するかどうかを設定します。

- デフォルト : オフ

### デフォルトエリアのパススルー設定
ここに指定したエリア内に、デフォルトエリアの音声が聞こえるようになります。

#### ブラックリストを使用する
有効化するとパススルー設定がブラックリストになります。

すべてのエリアに対しデフォルトエリアの音声が聞こえるようになり、指定したエリア内のみ聞こえなくなります。

- デフォルト : オフ

#### パススルーエリア設定
デフォルトエリア内の音声が聞こえるエリアを指定できます。
